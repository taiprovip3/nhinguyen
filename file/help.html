LỖI PLUGIN

    Thông thường khi bạn chỉ bỏ plugins vào folder server và ko dán cả file của mình thì có thể xảy ra lỗi ngoài ý muốn nên các bạn tải file mình về hãy nén và dùng tất cả đừng chỉ lấy mỗi folder plugin hoặc folder world. Còn trường hợp lỗi plugin gây đứng console do plugin đó được dev nâng cấp và ko cho sử dụng free nữa các bạn hãy chú ý kỹ console xem nó dừng ở tên plugin nào và xoá nó đi.
